What Is Bijection?

Bijection is a method to count the number of elements in a set by counting the number of elements in a different set in which every single element corresponds to exactly one element in the first set, and every single element in the first set corresponds to exactly one element in the second set. Basically, we use this by finding a simpler set that has this one-to-one correspondence with the original set and count the number of elements in that set to find the number of elements in the original set.

Stars and Bars

One extremely useful method of bijection is known as stars and bars. Suppose we are asked to find the number of ways to distribute \(n\) identical balls amongst \(k\) boxes. It doesn't take too much thinking to realize that casework will be extremely difficult and tedious if \(n\) and \(k\) are large. Instead, we use bijection. Rather than distributing the balls amongst the boxes, we can distribute dividers between the balls, with the spaces between dividers representing the number of balls within the boxes. The set of ways to distribute the dividers, bars, amongst the balls, stars, has a one-to-one correspondence with the original set of ways to distribute balls amongst boxes and is far easier to count. All we have to do is choose which of the \(n+k-1\) objects (stars and bars) are bars. That is, if we have no restrictions on the number of balls in each box. If each box must have at least one ball, each pair of bars must have at least one star between the bars. Therefore, we choose the bars (\(k-1\)) from the spaces between the stars (\(n-1\)).
